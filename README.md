# 📚 Digital Library — Платформа для обмена знаниями

**Проблема:** Студенты и учителя часто теряют важные учебные материалы в бесконечных чатах и мессенджерах.

**Решение:** Наше приложение предоставляет единую структурированную базу данных для хранения и быстрого поиска учебных ресурсов.

## ✨ Основные функции
- 🔍 **Умный поиск:** Мгновенный поиск по названию, автору или категории.
- ➕ **Добавление ресурсов:** Удобная форма (модальное окно) для пополнения библиотеки.
- 📱 **Адаптивный дизайн:** Сайт отлично работает как на ПК, так и на смартфонах.
- 🛠 **SQL Integration:** Надежное хранение данных через SQLAlchemy.

## 🚀 Технологический стек
- **Backend:** Flask (Python)
- **Database:** SQLite / SQLAlchemy
- **Frontend:** HTML5, CSS3, Bootstrap 5, FontAwesome

## 🛠 Установка и запуск (локально)
1. Клонировать репозиторий:
   `git clone https://github.com/sazmak/ed-Tech-Challange-SAFARI-team.git`
2. Создать виртуальное окружение и установить зависимости:
   `pip install -r requirements.txt`
3. Запустить проект:
   `python run.py`
